what is useeffect and why we use useEffect.?

use effect is also a hook in react we use uesEffect to handle the side effects like fetching the data inside the component without useEffect it will run on every render which can cause unnecessary API calls and re-renders. useEffect allows fetching the data only once when the component mounts. this improves performance by reducing unnecessary API calls, and making the component more effective
